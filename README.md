Freeze 'Em

Created by: bydrabokin1755
Email: bydrabokin1755@gmail.com

GitHub Repository: https://github.com/Nacho-games/freeze-em

This game was created for the Winter Game Jam of 2026

In freeze them you are a scared snowman that is constantly attacked by aliens.
To defend yourself from those alines you can freeze them with snowballs and kill them with you hammer for more snowballs.
There is also somepowerups to help you survive.

I have include the source code along the assets as well as executables for Windows, Linux and MacOS.
You can always run the source code directly inside of source-code (cd freezeThem/source-code/---- python3 main.py)

Controls:
WASD/arrow keys: Move
Mouse: aim snowball
Mouse click: throw snowball
Space: Hammer
1, 2, 3, 4 number keys: Powerups

All the graphical assets are created by me. But the audios and the font are open source.

If you find any bugs or have suggestions please comment it or send me an email.

I hope you like it and have fun!
